# Domain Name Brainstormer

This skill helps you find the perfect domain name by generating creative, memorable, and available options for your business or project.

## When to Use This Skill

- Launching a startup and need a domain name
- Creating a personal brand or portfolio site
- Naming a new product or service
- Rebranding an existing business
- Finding alternatives when your first choice is taken

## What This Skill Does

1. **Brainstorms Names**: Generates dozens of creative options based on your concept
2. **Checks Multiple TLDs**: .com, .io, .dev, .ai, .co, .app, .net
3. **Suggests Variations**: Prefixes, suffixes, portmanteaus, and creative misspellings
4. **Provides Ratings**: Scores names on memorability, brandability, and clarity
5. **Guides Decisions**: Recommends best options and explains trade-offs

## Instructions

### Step 1: Gather Information

Ask the user:
- What does the business/product do? (1-2 sentences)
- Target audience?
- Any keywords that must be included?
- TLD preference? (.com preferred, .io for tech, .ai for AI, etc.)
- Tone? (professional, playful, technical, creative)
- Budget? (premium domains can cost $$$)
- Any names to avoid or already checked?

### Step 2: Generate Name Categories

Generate 5-8 names per category:

**Descriptive Names** (clear what it does):
- Combine core function words
- Example: "CloudStore", "QuickSend"

**Invented/Portmanteau Names** (unique blend):
- Combine two relevant words
- Example: Pinterest (pin + interest), Instagram (instant + telegram)

**Metaphor Names** (use imagery):
- Abstract but memorable
- Example: Amazon, Apple, Stripe

**Action Names** (verb-based):
- What users DO with your product
- Example: Notion, Vercel, Deploy

**Personal/Founder Names** (if applicable):
- Good for consultancies and personal brands

### Step 3: Check Availability

Guide user to check:
1. **Namecheap** (namecheap.com) - Search multiple TLDs at once
2. **GoDaddy** (godaddy.com) - Bulk domain check
3. **Instant Domain Search** (instantdomainsearch.com) - Real-time results

```
Check these names at namecheap.com/domains/domain-name-search:
- [name1].com
- [name2].io  
- [name3].dev
```

### Step 4: Rate and Recommend

Score each name (1-5 stars) on:
- ✨ **Memorability**: Easy to remember after hearing once
- 🔤 **Spellability**: Easy to spell from hearing it
- 🎯 **Clarity**: Does it hint at what you do?
- 💼 **Brandability**: Does it feel like a brand?
- 📢 **Say-ability**: Easy to say out loud

### Step 5: Present Final Shortlist

```markdown
## My Top 5 Recommendations

1. **[Name].com** ⭐⭐⭐⭐⭐
   Why: [Explanation]
   Availability: Likely available / Check now

2. **[Name].io** ⭐⭐⭐⭐
   Why: [Explanation]
   ...
```

## TLD Guide

| TLD | Best for | Cost/year |
|-----|----------|-----------|
| .com | Everyone - most trusted | $10-15 |
| .io | Tech startups | $30-50 |
| .ai | AI products | $50-100 |
| .dev | Developer tools | $12-15 |
| .app | Mobile apps | $15-20 |
| .co | Companies | $25-30 |
| .net | Networks/tech | $12-15 |

## What Makes a Good Domain Name

✅ **Do**:
- Keep it under 15 characters
- Make it easy to spell phonetically
- Choose .com if possible
- Make it brandable (unique, distinctive)

❌ **Avoid**:
- Hyphens (looks spammy)
- Numbers (confusing: "4" or "four"?)
- Trademarked terms
- Hard-to-spell words

## Tips

- Check social media handle availability too (@handle)
- Domain squatters watch Google Trends - act fast on new ideas
- A great .io beats a terrible .com
- If your .com is taken, try adding "get", "use", "try" prefix
